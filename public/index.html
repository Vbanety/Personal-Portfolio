Habilidades

Venho absorvendo a percepção de estruturação e semântica das páginas com HTML e HTML5, criando a maturidade adequada para a utlização das TAGS com coerência, facilidando na elaboração de estruturaçãosfdsf.

A questão de estilização com CSS e CSS3, a percepção da correta utilização dos seletores com suas propriedades de acordo com a necessidade dos projetos, está sendo priorizado na construção dos meus projetos.

De acordo com a utilização frequente do JAVASCRIPT, usando condicionais, funções e manipulação do DOM, através de exercícios e utlização nos projetos, consigo vizualizar com mais facilidade a criação de uma lógica para criar uma solução.

A biblioteca NODE.JS derivada do Javascript, está dentro da minha rotina de estudos, ainda não estou utilizando com muita frequência, apenas por não ter uma boa base do próprio Javascript, mais está na minha lista de estudos.

O conceito de API e SQL, a manupulação de dados e interação com o Frontend, estou aderindo aos poucos nos meus projetos, até então o conhecimento a respeito, é através do acompanhamento de bootcamps, tutoriais e documentação ou blogs.

Estou consolidando toda a base de HTML, CSS e JAVASCRIPT para compreender com mais facilidade o framework REACT, para manipulação dos Components e correta utilzação das Props.
